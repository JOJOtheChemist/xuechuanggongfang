# 高考AI联网搜索 Smoke 测试报告

测试时间：2026-05-20

测试目标：
- 验证 `kode-sdk-xuechuang` 中新接入的 `web_search` / `web_fetch` 是否可用于高考 AI 的联网文本搜索
- 验证常见学校咨询问题是否能搜到可用的文字结果
- 验证高考 AI 本体是否能在对话中触发联网工具

测试范围：
- 学校就业情况
- 宿舍环境
- 周围小吃街情况
- 学校周围交通
- 男女比例

测试样例学校：
- 昆明冶金高等专科学校

## 一、当前联网工具状态

已接入工具：
- `web_search`
- `web_fetch`

当前实现特点：
- 只做文字网页搜索，不做图片搜索
- 搜索源使用 `Bing RSS + Sogou`
- 会优先保留和“学校名 + 招生/分数线/就业/宿舍/环境/交通”相关的结果
- `web_fetch` 会在 `https` 失败时回退尝试 `http`

## 二、直接工具 Smoke Test 结果

### 1. 学校就业情况

测试 query：
- `昆明冶金高等专科学校 就业怎么样 就业质量 毕业生就业`

搜索结果表现：
- 能搜到可用文字页
- 代表结果：
  - `昆明冶金高等专科学校 - 掌上高考`
  - `昆明冶金高等专科学校毕业生就业情况如何？`

正文抓取表现：
- `web_fetch` 成功抓到第三方文字页正文

抓取到的关键信息摘要：
- 毕业生深受用人单位欢迎
- 学校重视就业工作
- 推行“预就业制”“工学交替制”
- 近 50% 毕业生在昆明地区工作
- 连续五年就业率在 90% 以上

结论：
- `就业情况` 这类问题，当前联网工具可以搜到，而且能抓到足够支撑回答的文字内容

### 2. 宿舍环境

测试 query：
- `昆明冶金高等专科学校 宿舍 环境 食堂 校园`

搜索结果表现：
- 能搜到可用文字页
- 代表结果：
  - `校园生活-昆明冶金职业大学`
  - `昆明冶金高等专科学校宿舍条件,宿舍图片和环境空调及分配方法`

正文抓取表现：
- 官网 `校园生活` 页可成功抓正文
- 第三方宿舍页也可抓到正文，但编码表现不稳定

抓取到的关键信息摘要：
- 官网页能抓到两校区信息：
  - 莲华校区：昆明市学府路 388 号
  - 安宁校区：安宁市宁泊路 63 号
- 第三方页中可见：
  - 宿舍条件很好
  - 食堂评价有提及
  - 安宁校区环境不错

结论：
- `宿舍环境` / `校园环境` 这类问题能搜到
- 但这类问题比就业、分数线更依赖第三方页面质量，结果稳定性中等

### 3. 周围小吃街情况

测试 query：
- `昆明冶金高等专科学校 周围 小吃街 美食 商圈`

搜索结果表现：
- 能搜到结果，但结果较弱
- 当前命中页主要是：
  - 一份关于“学校附近小吃摊饮食看法”的问卷页

正文抓取表现：
- `web_fetch` 成功抓到问卷正文

抓取到的关键信息摘要：
- 明确出现“学校附近小吃摊饮食”的讨论
- 问卷内容涉及：
  - 是否喜欢去路边摊
  - 小吃是否卫生
  - 是否造成交通、市容、健康影响

结论：
- `周围小吃街情况` 可以搜到一些相关文本线索
- 但目前更容易搜到问卷、讨论页，而不是结构化“美食街攻略”
- 这类问题可回答，但信息丰富度一般

### 4. 学校周围交通

测试 query：
- `昆明冶金高等专科学校 周围 交通 地铁 公交 出行`

搜索结果表现：
- 能搜到官网相关文字页
- 代表结果：
  - `通知公告-昆明冶金高等专科学校`
  - `关于昆明冶金高等专科学校“校园直通车”运行通知`

正文抓取表现：
- 官网交通通知页可成功抓正文

抓取到的关键信息摘要：
- 抓到“昆明冶专教职工交通车运行时刻表”
- 可见始发地点、终点、运行周期
- 文中出现：
  - 莲华校区
  - 安宁校区
  - 安宁公交
  - 学校值班车

结论：
- `周围交通` / `出行` 类问题可以搜到，并且是可回答的
- 尤其是校车、校区通勤、公交相关信息，文本结果可用

### 5. 男女比例

测试 query：
- `昆明冶金高等专科学校 男女比例 男女生比例`

搜索结果表现：
- 当前结果基本不可用
- 返回了昆明旅游、百科类页面
- 没有找到可靠的学校男女比例数据页

结论：
- `男女比例` 这类问题，目前不能稳定搜到可信文本结果
- 如果用户这样问，AI 应该更谨慎：
  - 明确说明“未检索到可靠公开统计”
  - 不要编造

## 三、分数线补充验证

虽然你这轮重点不是分数线，但为了验证工具整体能力，也补测了：

测试 query：
- `昆明冶金高等专科学校 录取分数线 招生网`

结果表现：
- 搜到学校招生网
- 搜到历年招生页
- 搜到掌上高考汇总页
- 搜到官网具体分数线页

正文抓取表现：
- 成功抓到官网 `2022年昆明冶金高等专科学校分省分类别录取分数线`

抓取到的关键信息摘要：
- 有明确年份
- 有省份
- 有科类
- 有最高分、最低分、平均分

结论：
- `录取分数线` 问题当前最好用，联网工具可稳定支持

## 四、对高考 AI 本体的 Smoke 测试

目标：
- 验证高考 AI 是否能通过 `/chat/sync` 正常启动对话
- 验证其是否能在对话中触发联网工具

测试结果：
- 未完成对话级 smoke

原因不是联网工具本身，而是本地服务环境阻塞：
- 启动 `server/index.ts` 时，服务在数据库初始化阶段失败
- 报错信息：
  - `role "xuechuang_workshop" does not exist`

影响：
- 本地 `chat/sync` 服务未成功启动
- 因此无法完成“高考 AI 在对话中真实触发 tool”的最终验证

当前结论：
- `工具级 smoke` 已经完成，证明联网工具可用
- `Agent 对话级 smoke` 被本地数据库环境阻塞，暂未证明

## 五、综合判断

从当前结果看，联网搜索工具对这些问题的支持程度如下：

### 可较好支持

- 学校就业情况
- 录取分数线
- 学校交通 / 校车 / 公交相关信息

### 可支持，但质量中等

- 宿舍环境
- 校园环境
- 周围小吃街 / 附近饮食

### 当前不可靠

- 男女比例

## 六、建议

### 1. 对话策略建议

高考 AI 遇到这类问题时，建议这样处理：
- `就业情况`：优先搜“就业质量报告 / 毕业生就业 / 掌上高考 / 学校就业网”
- `宿舍环境`：优先搜“校园生活 / 宿舍 / 食堂 / 校区”
- `周围交通`：优先搜“校车 / 公交 / 校区地址 / 校园直通车”
- `周围小吃街`：可搜“学校附近 小吃 / 美食 / 商圈”，但需接受信息可能偏弱
- `男女比例`：若无可靠公开数据，应明确说明“未检索到可靠来源”

### 2. 工具层建议

还可以继续优化：
- 进一步剔除搜狗特殊聚合结果里的非文本噪音
- 对“学校名必须命中”再加强一点
- 为“就业 / 环境 / 交通 / 周边生活”分别做更细的 query 模板

## 七、最终结论

如果问题是：
- `学校就业情况`
- `宿舍环境`
- `周围小吃街情况`
- `学校周围交通`

那么当前联网工具总体上是可以搜到文字结果的，其中：
- `就业情况`
- `交通`
- `分数线`

效果最好。

如果问题是：
- `男女比例`

当前不能稳定搜到可靠公开数据，不能承诺可答。

另外：
- 当前已经证明 `web_search + web_fetch` 工具链可用
- 但“高考 AI 本体在真实对话里是否成功触发这些联网 tool”，还需要先修复本地数据库角色问题，再做一次 `/chat/sync` 级 smoke
